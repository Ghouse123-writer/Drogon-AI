# Drogon AI Assistant 🐉

A Python desktop AI assistant inspired by Game of Thrones.

## Features

- Voice wake word ("Hello")
- Text mode
- Open Chrome, YouTube, Spotify, Notepad, Calculator, Paint, CMD
- Web search
- YouTube search
- OpenAI-powered MUN speech generation
- Saves generated speeches to speech.txt

## Installation

```bash
py -3.12 -m pip install -r requirements.txt
```

## Set API Key

```bash
setx OPENAI_API_KEY "your_api_key_here"
```

Close and reopen Command Prompt after running the command.

## Run

```bash
py -3.12 drogon.py
```

## Example Commands

### Text Mode
- open chrome
- search latest AI news
- youtube python tutorial
- speech cyber warfare and international security

### Voice Mode
1. Say: "Hello"
2. Then say a command such as "Open Chrome" or "Speech democracy and digital governance"
